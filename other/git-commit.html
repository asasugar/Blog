# 推荐使用 cz-git

这种提交格式不香吗？ 😍

# 一、全局安装

npm install -g commitizen cz-git
1

run

git cz
1

# 二、本地安装

npm install commitizen cz-git -D
1

package.json:

{
  "config": {
    "commitizen": {
      "path": "cz-git"
    }
  },
}
